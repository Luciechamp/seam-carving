SEAM CARVING

Projet permettant de redimensionner une image en utilisant l'algorithme du Seam carving.
https://dl.acm.org/doi/abs/10.1145/1435417.1435437

Appel :
$ javac SeamCarving.java
$ java SeamCarving NOM_IMAGE POURCENTAGE_HAUTEUR POURCENTAGE_LARGEUR

Exemple :
$ java SeamCarving aphid.jpg 20 50
